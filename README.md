# Wood Frame

Sitio web oficial de Wood Frame, una marca colombiana de marcos de gafas artesanales en bambú.

## Sitio

El proyecto es una página estática construida con HTML, CSS y JavaScript. Incluye visualizadores interactivos para los modelos 3D de los productos y un formulario de cotización conectado directamente con WhatsApp.
